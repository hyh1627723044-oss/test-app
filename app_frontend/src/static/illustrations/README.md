# Illustration Assets

These PNG files are replaceable hand-drawn illustration slots for the mini program.

| File | Used for |
| --- | --- |
| `search-cat.png` | Search card mascot on the recipe home page |
| `my-recipes-pot.png` | "My recipes" entry on the profile page |
| `tab-recipes.png` | Bottom navigation recipe/menu icon |
| `tab-plan.png` | Bottom navigation meal-plan icon |
| `tab-my.png` | Bottom navigation profile icon |
| `empty-recipes.png` | Empty recipe/search result state |
| `empty-plan.png` | Empty meal-plan slot state |
| `upload-recipe.png` | Recipe cover upload guide |
| `ai-chef.png` | Future AI recommendation entry |
| `plan-success.png` | Future add-to-plan success feedback |
| `icon-recipe-add.png` | Add-recipe shortcut on the recipe home page |

Recipe cards with no cover image use a WXSS-rendered first-character placeholder instead of a fixed PNG.

Keep the filenames stable when replacing the artwork so WXML references do not need to change.
